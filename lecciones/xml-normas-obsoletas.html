En este páxina se listan as versións obsoletas de as recomendacións relacionadas con XML (XML, XSL, XQuery, etc.) e a web semántica (RDF, OWL, SPARQL, etc.). as últimas versións se comentan nas seccións Normas e recomendacións XML e Normas e recomendacións Web semántica.

Recomendacións XML obsoletas

XML Core

O grupo de traballo XML Core publicou as seguintes recomendacións que fueron sustituidas por versións posteriores:

  • febreiro de 1998: XML 1.0, editada por Tim Bray e outros.
  • xaneiro de 1999: Namespaces in XML 1.0, editada por Tim Bray e outros.
  • xuño de 1999: Associating Style Sheets with XML documents 1.0, editada por James Clark.
  • outubro de 2000: XML 1.0 (2ª edición), editada por Tim Bray e outros.
  • marzo de 2001: Canonical XML 1.0, editada por John Boyer.
  • xuño de 2001: XML Base, editada por Jonathan Marsh.
  • xuño de 2001: XML Linking Language (XLink) 1.0, editada por Steve DeRose e outros.
  • outubro de 2001: XML Information Set, editada por John Cowan e Richard Tobin.
  • febreiro de 2004: XML 1.0 (3ª edición), editada por Tim Bray e outros.
  • febreiro de 2004: XML 1.1, editada por Tim Bray e outros.
  • febreiro de 2004: Namespaces in XML 1.1, editada por Tim Bray e outros.
  • decembro de 2004: XML Inclusions (XInclude) 1.0, editada por Jonathan Marsh e David Orchard.
  • agosto de 2006: XML 1.0 (4ª edición) editada por Tim Bray e outros.
  • agosto de 2006: Namespaces in XML 1.0 (2ª edición), editada por Tim Bray e outros.
  • abril de 2010: Associating Schemas with XML documents 1.0, editada por Paul Grosso e Jirka Kosek.
  • agosto de 2011: Associating Schemas with XML documents 1.0 (2ª edición), editada por Paul Grosso e Jirka Kosek.

O grupo de traballo XML Core publicou as seguintes notas do grupo de traballo:

  • outubro de 2003: LBase: Semantics for Languages of the Semantic Web, editada por R.V.Guha e Patrick Hayes.
  • outubro de 2012: Associating Schemas with XML documents 1.0 (3ª edición), editada por Paul Grosso e Jirka Kosek.
  • xullo de 2016: XML Inclusions (XInclude) 1.1, editada por Jonathan Marsh e outros

    Esta nota do grupo de traballo es idéntica a a norma ISO/IEC 19757-11. O subcomité ISO/IEC JTC 1/SC 34 se encarga da norma ISO 19757 (DSDL: Document Schema Definition Languages) que trata de varios linguaxes de definición de documentos (DTD, Schema, RELAX NG, Schematron, etc.). Algunas partes de esta norma se encuentran disponibles.

Al cerrarse o grupo de traballo XML Core, varios proyectos de recomendacións quedaron abandonados e ni siquiera se publicaron como notas do grupo de traballo:

  • xaneiro de 2010: XML Schema Definition Language (XSD): Component Designators.
  • marzo de 2013: XML Normalization.
