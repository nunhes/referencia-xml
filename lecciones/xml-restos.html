Outros

Tipo Expresión Ejemplo Significado
Conectores , (nombre1, nombre2) hay dos elementos seguidos
| (nombre1 | nombre2) hay uno dos dos elementos
Repetición (nombre1) el elemento aparece sólo unha vez
? (nombre1?) el elemento no aparece o aparece sólo unha vez
* (nombre1?) el elemento no aparece o aparece unha o máis veces
+ (nombre1+) el elemento aparece unha o máis veces
Agrupación () los paréntesis permiten agrupar expresións para crear expresións máis complejas
