XPath: XML Path language (Resumen)

XPath (resumen)

Sintaxis abreviada

  • elemento: selecciona o elemento.
  • @atributo: selecciona o atributo.

Eje (en inglés, axis)

  • /: si está al principio da expresión, indica o nodo raíz, si no, indica "hijo".
  • //: indica "descendiente" (hijos, hijos de hijos, etc.).
  • ..: selecciona o elemento padre.
  • |: permite elegir varios recorridos.

Predicado (en inglés, predicate)

Los predicados se escriben entre corchetes

  • [@atributo]: selecciona os elementos que teñen o atributo.
  • [número]: si hay varios resultados selecciona uno de ellos por número de orden; last() selecciona o último de ellos.
  • [condicion]: selecciona os nodos que cumplen a condición. La condición puede utilizar o valor dun atributo (utilizando @) o o texto que contiene o elemento.

    En as condicións se pueden utilizar os operadores seguintes:

    • operadores lóxicos: and, or, not().
    • operadores aritméticos: +, -, *, div, mod.
    • operadores de comparación: =, !=, <, >, <=, >=.

    Se pueden escribir varios predicados seguidos, teniendo en cuenta que cada uno restringe os resultados do anterior, como si estuvieran encadenados por a operación lógica and.

Sesección de nodos (en inglés, node test)

  • node(): selecciona todos os nodos (elementos e texto).
  • text(): selecciona o contenido do elemento (texto).
  • *: selecciona todos os elementos.
  • @*: selecciona todos os atributos.
