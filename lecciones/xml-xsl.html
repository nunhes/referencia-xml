XSL: Linguaxe de Hojas de Estilo Extensible

XSL

Aunque as follas de estilo CSS se pueden aplicar a os documentos XML, as follas de estilo teñen muchas limitacións xa que se crearon para complementar al HTML, es decir, para ver páxinas web en pantallas de ordenador. da misma manera que o XML es unha generalización da HTML, o W3C creó unha generalización de as follas de estilo CSS a a que se denominó XSL (eXtensible Stylesheet Language), es decir, Linguaxe de follas de estilo extensible.

O W3C ha desenvolvedo tres linguaxes:

  • XPath: un linguaxe para referirse a partes dun documento XML.
  • XSLT (XSL Transformation, es decir, Transformacións XSL): un linguaxe para transformar documentos XML.
  • XSL-FO (XSL Formatting Objects, es decir, Objetos de formato XSL): un linguaxe para especificar o formato dun documento XML e posteriormente convertirlo a PDF o PostScript.
