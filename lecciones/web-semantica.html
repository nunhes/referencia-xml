A Web Semántica

A Web Semántica foi unha iniciativa do W3C creada en 2001 co obxectivo de que os datos disponibles na Web pudieran ser procesados de forma automática. A idea era que a web de entonces, formada por documentos, se convirtiera en unha web formada por datos con significado e que as máquinas pudieran procesar toda esa información.

Con esa idea se desenvolveron toda unha serie de tecnoloxías (RDF, OWL, SPARQL, GRDDL, etc.) que permitieran representar, codificar e consultar a información de forma automática.

Los grupos de traballo creados por o W3C para desenvolver estas tecnoloxías aprobaron unha primera generación de recomendacións antes de 2010 e no caso de as principales tecnoloxías aprobaron unha segunda versión antes de 2014, pero os grupos se han ido cerrando unha vez cumplida esa misión.

A imagen seguinte muestra as fechas de aprobación de as recomendacións relacionadas con as principales tecnoloxías da web semántica (las dos últimas líneas muestran tecnoloxías menos importantes).

Desgraciadamente, con o paso do tiempo o interés por a web semántica ha ido decayendo. as tecnoloxías desenvolvedas para a web semántica se empregan en moitos sectores e se han integrado no arsenal de técnicas que se engloban bajo o término Inteligencia Artifical, pero no teñen o papel central que esperaba o W3C.

En decembro de 2013 os grupos de traballo dedicados a a web semántica se integraron na actividad do W3C Data, pero simplemente como grupo de interés (es decir, sin o objetivo de producir recomendacións). Con o paso do tiempo se han vuelto a crear grupos de traballo estables, dedicados al tratamiento e intercambio de datos con ferramentas da web semántica.

A posteriori, es fácil reconocer que o objetivo da web semántica era demasiado ambicioso e que as ferramentas propuestas no eran suficientes para poder conseguir su objetivo. Pero tampoco podemos olvidar que a base da web semántica era o XML e por lo menos desde 2003 a web realmente existente se ha ido alejando cada vez máis do XML, lo que complicaba na práctica todo o proyecto.

Resource Description Framework (RDF)

RDF es un modelo de datos para metadatos, diseñado para representar a información disponible na web. as expresións RDF teñen a forma sujeto-predicado-objeto (por eso se llaman tripletes).

Primer grupo de traballo RDF no W3C: Homepage - Charter: 2001-2002 - 2002-2004

Segundo grupo de traballo RDF no W3C: Homepage - Charter: 2011-2014

O primer grupo de traballo RDF se debió crear en 1997 e se cerrou en maio de 2004. O segundo grupo de traballo RDF se creó en febreiro de 2011 e se cerrou en xullo de 2014.
