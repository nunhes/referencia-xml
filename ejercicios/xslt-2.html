Estes exercicios corresponden ao visto na sección follas de estilo XSLT.

En cada exercicio, a partir dun documento XML, se trata de obter diferentes documentos XML escribindo diferentes fojas de estilo XSLT. É aconsellable crear unha copia do documento XML para cada apartado e que cada copia inclua o enlace a unha folla de estilo XSLT diferente.

Podes consultar unhas posibles solucións destes exercicios.

XSLT (2) - Exercicio 1

Dado o seguinte documento XML, crea as follas de estilo XSLT que devolvan a resposta desexada.

<?xml version="1.0" encoding="UTF-8"?>
<museos>
  <museo>
    <nombre>Museo do Prado</nombre>
    <ciudad>Madrid</ciudad>
    <pais>España</pais>
  </museo>
  <museo>
    <nombre>British Museum</nombre>
    <ciudad>Londres</ciudad>
    <pais>Reino Unido</pais>
  </museo>
  <museo>
    <nombre>National Gallery</nombre>
    <ciudad>Londres</ciudad>
    <pais>Reino Unido</pais>
  </museo>
</museos>

XSLT (2) - Exercicio 1-1

Convertir as etiquetas en atributos:

<?xml version="1.0" encoding="UTF-8"?>
<museos>
  <museo nombre="Museo do Prado" ciudad="Madrid" pais="España"/>
  <museo nombre="British Museum" ciudad="Londres" pais="Reino Unido"/>
  <museo nombre="National Gallery" ciudad="Londres" pais="Reino Unido"/>
</museos>
