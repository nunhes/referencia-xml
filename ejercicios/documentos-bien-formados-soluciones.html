Se ofrecen a continuación unhas posibles solucións dos ejercicios de Documentos ben formados. Estos exercicios se pueden resolver de varias maneras. En as solucións seguintes se proponen algunhas posibilidades, pero outras son aceptables.

WFD - Exercicio 1 - Deportistas - Solución

  • En as etiquetas <deporte> (líneas 4 e 7) aparecen os términos Atletismo e Natación sueltos. 
    Error at line 4, column 24: not well-formed (invalid token)

    Se podría corregir escribiendo esos términos como valores dun atributo: 

        <deporte nombre="Atletismo" />
 ...
    <deporte nombre="Natación" />

    o como texto dentro da etiqueta:

        <deporte>Atletismo</deporte>
 ...
    <deporte>Natación</deporte>
  • La primera etiqueta <deportista> no está cerrada. 
    Error at line 10, column 3: mismatched tag

    Se podría corregir cerrándola: 

      <deportista>
    <deporte Atletismo />
    <nombre>Jesse Owens</nombre>
  </deportista>
  <deportista>
 ...
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<deportistas>
  <deportista>
    <deporte nombre="Atletismo" />
    <nombre>Jesse Owens</nombre>
  </deportista>
  <deportista>
    <deporte nombre="Natación" />
    <nombre>Mark Spitz</nombre>
  </deportista>
</deportistas>

WFD - Exercicio 2 - Películas - Solución

  • No hay unha etiqueta raíz que englobe o resto de etiquetas. 
    Error at line 6, column 1: junk after document element

    Se podría corregir añadiendo unha etiqueta raíz, cuyo nombre puede ser cualquiera, mientras no coincida con otra etiqueta, por ejemplo <peliculas>: 

    <?xml version="1.0" encoding="UTF-8"?>
<peliculas>
  <pelicula>
    <titulo>Con faldas e a lo loco</titulo>
 ...
  <autor />barto</autor>
</peliculas>
  • La primera etiqueta <autor> tiene a barra de as etiquetas vacías, pero luego hay unha etiqueta de cierre. 
    Error at line 11, column 17: mismatched tag

    Se podría corregir borrando a barra: 

    <autor>barto</autor>
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<peliculas>
<pelicula>
  <titulo>Con faldas e a lo loco</titulo>
  <director>Billy Wilder</director>
</pelicula>
<pelicula>
  <director>Leo McCarey</director>
  <titulo>Sopa de ganso</titulo>
</pelicula>
<autor>barto</autor>
</peliculas>

WFD - Exercicio 3 - Texto - Solución

  • Las etiquetas de apertura e cierre no coinciden 
    Error at line 3, column 38: mismatched tag

    Se podría corregir modificando a etiqueta de apertura o a de cierre, por ejemplo: 

      <titulo>XML explicado a os niños</titulo>
  • Los caracteres :// no están permitidos nas etiquetas 
    Error at line 8, column 10: not well-formed (invalid token)

    Se podría corregir eliminando esos caracteres, por ejemplo: 

       <http>www.example.org</http>
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<texto>
  <titulo>XML explicado a os niños</titulo>
  <párrafo>O <abreviatura>XML</abreviatura>define cómo crear
  linguaxes de marcas.</párrafo>
  <párrafo>Las marcas se añaden a un documento de texto
  para añadir información.</párrafo>
  <http>www.example.org</http>
</texto>

WFD - Exercicio 4 - Información geográfica - Solución

  • O nombre de as etiquetas no pueden contener espacios: 
    Error at line 2, column 19: not well-formed (invalid token)

    Se podría corregir modificando a etiqueta de apertura o a de cierre, por ejemplo: 

     <geografia_mundial>
 ...
</geografia_mundial>

    o

     <geografia-mundial>
 ...
</geografia-mundial>
  • La etiqueta de cierre no puede contener atributos: 
    Error at line 6, column 24: not well-formed (invalid token)

    Se podría corregir escribiendo o valor no interior da etiqueta, por ejemplo: 

    <capital>Madrid</capital>
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<geografia-mundial>
  <pais>
    <pais>España</pais>
    <continente>Europa</continente>
    <capital>Madrid</capital>
  </pais>
</geografia-mundial>

WFD - Exercicio 5 - Programas - Solución

  • Los atributos no se pueden repetir: 
    Error at line 3, column 45: duplicate attribute

    Se podría corregir uniendo os atributos nun único atributo, por ejemplo: 

       <programa nombre="Firefox" licencia="GPL MPL" />

    o duplicando as etiquetas, por ejemplo:

       <programa nombre="Firefox" licencia="GPL" />
   <programa nombre="Firefox" licencia="MPL" />
  • Los valores dos atributos deben ir entre comillas: 
    Error at line 4, column 46: not well-formed (invalid token)

    Se podría corregir añadiendo comillas, por ejemplo: 

        <programa nombre="Inkscape" licencia="GPL" />
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<programas>
  <programa nombre="Firefox" licencia="GPL MPL" />
  <programa nombre="LibreOffice" licencia="LGPL" />
  <programa nombre="Inkscape" licencia="GPL" />
</programas>

WFD - Exercicio 6 - Mundiales de fútbol - Solución

  • Las etiquetas necesitan tener un nombre de etiqueta: 
    Error at line 4, column 10: not well-formed (invalid token)

    Se podría corregir añadiendo un nombre de atributo, por ejemplo: 

       <pais nombre="España" />

    o escribiendo o valor como contenido da etiqueta, por ejemplo:

       <pais>España</pais>
  • Los nombres de etiquetas no pueden empezar por un número: 
    Error at line 5, column 6: not well-formed (invalid token)

    Se podría corregir añadiendo un nombre de etiqueta, por ejemplo: 

         <ano valor="1982" />

    o escribiendo o valor como contenido de unha etiqueta, por ejemplo:

       <ano>1982</ano>
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<mundiales-de-futbol>
  <mundial>
    <pais>España</pais>
    <ano>1982</ano>
  </mundial>
</mundiales-de-futbol>

WFD - Exercicio 7 - Medios de transporte - Solución

  • O carácter inferior a (<) sólo puede utilizarse como inicio de etiqueta: 
    Error at line 3, column 26: not well-formed (invalid token)

    Se podría corregir escribiendo a entidad de carácter correspondiente, por ejemplo: 

      <bicicleta velocidad="v&lt;100km/h" />
  • Los nombre de atributos no pueden contener espacios: 
    Error at line 4, column 23: not well-formed (invalid token)

    Se podría corregir modificando o nombre do atributo, por ejemplo:

      <patinete velocidad-maxima="50 km/h"
  • Las etiquetas deben terminar por o carácter superior a (>): 
    Error at line 5, column 1: not well-formed (invalid token)

    Se podría corregir añadiendo os caracteres /> finales, por ejemplo: 

      <patinete velocidad-maxima="50 km/h" />
  • Una posible solución sería entonces: 
    <?xml version="1.0" encoding="UTF-8"?>
<mediosDeTransporte>
  <bicicleta velocidad="v&lt;100km/h" />
  <patinete velocidad-maxima="50 km/h" />
</mediosDeTransporte>
