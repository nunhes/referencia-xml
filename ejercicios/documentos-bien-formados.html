Estos exercicios corresponden a lo explicado na sección O que é XML.

Los seguintes documentos no están ben formados porque cada uno contiene dos errores (dos errores do mismo tipo cuentan como uno sólo). Corrija os errores e compruebe con XML Copy Editor que xa son documentos ben formados (si para corregir algún error hay que inventarse unha etiqueta o atributo, utilice un nombre que tenga relación con a información contida no documento).

Aunque en XML Copy Editor no es necesario guardar os documentos para comprobar que están ben formados, se recomienda guardarlos para poder consultarlos no futuro. O nombre do archivo podría ser wdf_XX.xml, donde XX es o número do exercicio.

Se pueden consultar as solucións de estos exercicios.

WFD - Exercicio 1 - Deportistas

<?xml version="1.0" encoding="UTF-8"?>
<deportistas>
  <deportista>
    <deporte Atletismo />
    <nombre>Jesse Owens</nombre>
  <deportista>
    <deporte Natación />
    <nombre>Mark Spitz</nombre>
  </deportista>
</deportistas>
