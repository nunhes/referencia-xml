Presentación

Me llamo Bartolomé Sintes Marco. Soy profesor de Informática no Instituto de Educación Secundaria Abastos de Valencia (España).

En o curso 2017/18 dediqué o final do tercer trimestre do curso (unas 12 horas de clase) a enseñar XML a alumnos do módulo Linguaxe de Marcas e Sistemas de Gestión da Información de 1º curso do ciclo formativo de grado superior Administración de Sistemas Informáticos en Red.

Desde entonces no he vuelto a enseñar XML en dicho módulo, excepto unha o dos clases que dedico a enseñar a sintaxis XML para poder trabajar con ficheros SVG. He seguido actualizando a información sobre os grupos de traballo do W3C, pero no as explicacións ni os exercicios.

Este curso estaba orientado a alumnos con conocimientos básicos de:

  • Diseño de páxinas web HTML e CSS, adquiridos no primer trimestre de este mismo módulo Linguaxes de Marcas e Sistemas de Gestión de Información.
    A título informativo se ofrecen materiales didácticos de ese módulo: https://www.mclibre.org/consultar/htmlcss/.
  • Programación básica, adquiridos no segundo trimestre de este mismo módulo Linguaxes de Marcas e Sistemas de Gestión de Información.
    A título informativo se ofrecen materiales didácticos de ese módulo: https://www.mclibre.org/consultar/python/

Los materiales didácticos de estos dos cursos sobre HTML/CSS e Programación básica sí que os he seguido actualizando e he ido eliminando en ellos as referencias al XML a medida que se han ido quedando obsoletas.

Formato de estos apuntes

  • Las páxinas de este curso son accesibles desde a páxina principal dos apuntes.
  • La páxina principal está dividida en cinco apartados:
    • no apartado Leccións XML se encuentran as seccións sobre conceptos básicos de XML que se abordan en este curso.
    • no apartado Exercicios se encuentran exercicios sencillos de XML agrupados por temas abordados nas seccións anteriores. Se incluyen tamén solucións dos exercicios propuestos.
    • no apartado Exámenes se encuentran ejemplos de exámenes de anos anteriores que se pueden utilizar como exercicios adicionales (algunos exercicios de exámenes se han incluido posteriormente en os exercicios). No se incluyen solucións dos exámenes propuestos.
    • no apartado Software se encuentran as seccións dedicadas a XML Copy Editor e outros programas.
    • no apartado Outros se encuentran seccións de referencia o material adicional al curso.
  • En a parte superior izquierda de cada páxina do curso se encuentra algúns elementos comúns:
    • : enlace a a páxina principal do curso.
    • : enlace al Principio da páxina actual (vuelve al Principio da páxina).
    • Un índice de apartados da sección que contiene enlaces a cada apartado. En o caso de as páxinas de exercicios puede haber tamén enlaces a as soluciones.
  • En a parte superior izquierda de cada páxina de examen se encuentra tamén o icono:
    • : enlace al índice de exámenes
  • En as explicacións se utilizan algúns iconos para completar os ejemplos de código:
    • : indica que o ejemplo es correcto.
    • : indica que o ejemplo es incorrecto.
    • : indica que o ejemplo no es incorrecto, pero está desaconsejado.
    • : enlace al fichero que contiene o ejemplo
  • En as explicacións sobre as etiquetas, atributos o propiedades se incluyen ejemplos en os que en su caso se indican as diferencias de comportamiento dos principales navegadores: Google Chrome, Firefox e Microsoft Edge.
    • Si os tres navegadores lo hacen correctamente e de manera similar, no se muestra ningún icono.
    • Si algún navegador lo hace incorrectamente, se muestran os iconos correspondientes:
      • : indica que o navegador lo hace correctamente
      • : indica que o navegador lo hace incorrectamente
      • : indica que no tengo claro si o navegador lo hace correctamente o no
  • Los ejemplos de ficheros xml están coloreados de forma similar a como se muestran no editor XML Copy Editor.

Programas empregados

En este curso se utilizarán programas libres multiplataforma, salvo en os casos que no sea posible hacerlo (por ejemplo, Microsoft Edge sólo está disponible en Windows).

Sistema operativo

Los ordenadores que utilizaron os alumnos no curso 2017/18 tenían instalado Windows 10, por lo que utilicé as versións para ese sistema operativo dos programas empregados, pero o curso puede seguirse perfectamente desde calquera distribución GNU/Linux.

Navegador

O navegador recomendado a os alumnos no curso 2017/18 fue Firefox, aunque o curso hace referencia tamén a Google Chrome e Microsoft Edge. Nos ordenadores dos alumnos estaban disponibles as últimas versións (que se iban actualizando automáticamente a versións posteriores).

En cdlibre.org hay unha sección dedicada a navegadores web libres, con información detallada sobre as últimas versións publicadas para Windows.

Editor

O editor que utilizaron os alumnos no curso 2017/18 fue XML Copy Editor, un sencillo editor libre e multiplataforma enfocado a a edición de archivos XML. También pudo utilizarse o editor Notepad++ (libre, pero sólo disponible en Windows), instalando o plug-in XML Tools.

En cdlibre.org hay unha sección dedicada a editores XML libres, con información detallada sobre as últimas versións publicadas para Windows.
